# Audit Reference Skill

参考資料との照合監査。

## プロセス

1. `C:\Users\bjex1\Desktop\OMS\reference\screenshot\` の画像一覧を取得
2. 対象画面ごとに:
   - リファレンス画像を Read
   - 対応する `src/app/.../page.tsx` を Read
   - 項目を抽出して比較
3. `oms-field-auditor` を呼んで詳細監査
4. 漏れリストを優先度付きで出力
5. 修正計画を `oms-form-master` または `oms-page-builder` に渡す

## 優先順位
高: マスタ登録系（顧客/卸先/仕入先/商品）
中: 設定系・受注/出荷詳細
低: ダウンロード系・サポート系

## 出力
- 監査レポート（漏れ項目リスト）
- 修正計画（どのページから直すか）
- 推定工数

## 重要
- 競合システム名は出力に書かない
- リファレンス画像 / 参考資料 と表現
